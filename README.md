# 🚀 Frontend Master Elite

> The Ultimate Frontend Engineering Power for Kiro IDE

## ⚡ What is this?

Frontend Master Elite is a **modular frontend intelligence system** designed to transform Kiro into a senior-level frontend engineering team.

It helps you:

- Build production-grade React & Next.js apps
- Design scalable UI architectures
- Enforce UI/UX best practices
- Apply motion design systems
- Maintain clean, scalable codebases
- Optimize performance and accessibility automatically

---

## 🧠 What it becomes

When activated, Kiro behaves like a full engineering team:

- Senior Frontend Engineer
- UI/UX Designer
- Software Architect
- Motion Designer
- Performance Engineer
- Accessibility Specialist
- QA Engineer

All working together.

⚙️ How it works

Frontend Master Elite is not a library.

It is a behavior system.

It works in 3 layers:

1. Steering (Intelligence)

Defines how Kiro thinks.

2. Prompts (Execution)

Defines what tasks are executed.

3. Templates (Structure)

Defines how projects are built.
